Como Fazer um Site

Introdução

Este guia básico irá te mostrar os passos fundamentais para criar um site simples usando HTML e CSS.

Fundamentos de HTML

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. É usado para estruturar o conteúdo da sua página.

Um documento HTML básico começa com a declaração do tipo de documento e inclui as tags principais: <html>, <head>, e <body>.

Fundamentos de CSS

CSS (Cascading Style Sheets) é usado para estilizar o conteúdo HTML. Com CSS, você pode controlar o layout, cores, fontes e outros aspectos visuais da sua página.

Um exemplo de código CSS inclui regras para estilizar elementos HTML, como se segue:

body {
    font-family: Arial, sans-serif;
    margin: 20px;
}
h1 {
    font-size: 30px;
    color: blue;
}
h2 {
    font-size: 20px;
    color: red;
}
p {
    font-size: 15px;
    font-family: "Times New Roman", serif;
    color: black;
}
